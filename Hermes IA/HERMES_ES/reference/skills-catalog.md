# Bundled Habilidads Catalog

Hermes ships with a large built-in habilidad library copied into `~/.hermes/habilidads/` on install. Each habilidad below links to a dedicated page with its full definition, setup, and usage.

Hermes also syncs bundled habilidads on `hermes update`, but the sync manifest respects local deletions and user edits. If a habilidad listed here is missing from your perfil's `~/.hermes/habilidads/` tree, it is still shipped with Hermes; restore it with `hermes habilidads reset <name> --restore`.

If a habilidad is missing from this list but present in the repo, the catalog is regenerated by `website/scripts/generate-habilidad-docs.py`.
